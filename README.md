Bulding Lick
============

Windows
-------

### Prerequisites:

- Visual Studio 2012 or newer (Express Edition is ok)

### Building

- Run `msbuild` in source directory

### Installation

- Copy `lick.exe` to whatever location is convenient

Linux
-----

### Prerequisites:

- GCC 4.7.x or newer (with working C++11 support)
- CMake 2.6 or newer

### Building

    $ cd source && cmake . && make
    
### Installation    

    $ make install
